# 🎮 Gaming Website (React JS Project)

A modern **Gaming Website UI** built using **React JS** and **React Router**.
This project demonstrates a multi-page layout with navigation between different sections like Games, Forums, Esports, Blog, FAQ, and more.

---

## 🚀 Features

* ⚛️ Built with **React JS**
* 🔗 Page navigation using **React Router**
* 🎨 UI components built using **React-Bootstrap**
* 📄 Multiple pages (Home, FAQ, Games, Forums, Esport, Blog, About, Contact)
* 🎨 Clean and modern UI
* 📱 Responsive design using **Bootstrap**
* ⚡ Fast development using **Vite**

## 🧠 Routing Logic

The application uses **React Router** to navigate between different pages.

Example from `App.jsx`:

```
<Route path="/" element={<Home />} />
<Route path="/faq" element={<FAQ />} />
<Route path="/games" element={<Games />} />
<Route path="/forums" element={<Forums />} />
<Route path="/esport" element={<Esport />} />
<Route path="/blog" element={<Blog />} />
<Route path="/about" element={<About />} />
<Route path="/contact" element={<Contact />} />
```

This allows users to move between different sections of the website without reloading the page.

## ▶️ Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/your-username/gaming-react-project.git
```

2️⃣ Go to project folder

```
cd gaming-react-project
```

3️⃣ Install dependencies

```
npm install
```

4️⃣ Start the development server

```
npm run dev
```

Now open:

```
http://localhost:5173
```
